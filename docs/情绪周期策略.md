# 情绪周期交易策略

## 一、情绪周期理论

市场情绪呈现周期性波动，分为五个阶段：**高潮期 → 退潮期 → 冰点期 → 回暖期 → 修复期**，循环往复。

### 周期判断指标

| 指标 | 计算方式 | 权重 |
|------|----------|------|
| 涨停数 | >80高潮, >50回暖, >30修复, <20冰点 | 40% |
| 连板高度 | 最高连板数 | 20% |
| 涨跌比 | 上涨家数/下跌家数 | 20% |
| 炸板率 | 炸板数/涨停数 | 20% |

---

## 二、各周期策略详解

### 1. 高潮期 🔥

**特征**：涨停数>80，连板高度>5，市场赚钱效应爆棚

**策略关注**：
| 操作 | 理由 |
|------|------|
| 追踪龙头连板 | 高潮期龙头效应明显，连板股溢价高 |
| 关注补涨机会 | 同板块滞涨股有补涨预期 |
| 控制仓位 | 高潮期后易出现分歧，需预留资金 |

**策略回避**：
| 操作 | 理由 |
|------|------|
| 追高接力 | 高位股获利盘多，容易被砸 |
| 满仓操作 | 情绪顶点风险大，需保留灵活性 |

**选股方向**：龙头股、连板股

---

### 2. 退潮期 📉

**特征**：涨停数下降，连板高度降低，炸板率上升

**策略关注**：
| 操作 | 理由 |
|------|------|
| 现金为王 | 退潮期亏钱效应强，保住本金最重要 |
| 等待信号 | 等待市场企稳信号，不急于抄底 |
| 小仓试错 | 可用小仓位试探市场温度 |

**策略回避**：
| 操作 | 理由 |
|------|------|
| 追涨停 | 退潮期涨停次日大概率低开，接力风险大 |
| 重仓操作 | 市场赚钱效应差，重仓亏损扩大 |
| 连板接力 | 连板高度受限，接力大概率被埋 |

**选股方向**：北向资金流入股、多因子高分股（防守配置）

---

### 3. 冰点期 ❄️

**特征**：涨停数<20，连板高度<3，市场极度恐慌

**策略关注**：
| 操作 | 理由 |
|------|------|
| 等待企稳 | 冰点期需等待止跌信号，不要盲目抄底 |
| 超跌优质股 | 优质股超跌后安全边际高，可分批布局 |
| 分批建仓 | 底部不确定，分批买入降低风险 |

**策略回避**：
| 操作 | 理由 |
|------|------|
| 恐慌杀跌 | 冰点期往往是底部区域，杀跌容易卖在最低 |
| 重仓抄底 | 底部可能反复磨，重仓抄底容易被套 |

**选股方向**：突破形态股、底部放量股（企稳信号）

---

### 4. 回暖期 🌱

**特征**：涨停数回升至30-50，开始出现连板

**策略关注**：
| 操作 | 理由 |
|------|------|
| 弱转强个股 | 前一日弱势今日转强，说明资金开始介入 |
| 首板确认 | 回暖期首板成功率高，关注封板质量 |
| 板块轮动 | 资金开始活跃，注意板块切换节奏 |

**策略回避**：
| 操作 | 理由 |
|------|------|
| 高位股 | 前期强势股可能补跌，风险大于机会 |
| 前期妖股 | 妖股炒作结束后容易持续阴跌 |

**选股方向**：弱转强股、首板股

---

### 5. 修复期 🔧

**特征**：涨停数50-80，市场逐步修复

**策略关注**：
| 操作 | 理由 |
|------|------|
| 超跌反弹 | 超跌股技术性反弹需求强，弹性大 |
| 技术金叉 | MACD/KDJ金叉确认底部，可靠性高 |
| 底部放量 | 放量说明资金开始进场，关注持续性 |

**策略回避**：
| 操作 | 理由 |
|------|------|
| 追涨 | 修复期反弹高度有限，追涨容易被套 |
| 重仓 | 市场方向不明，保持仓位灵活 |

**选股方向**：MACD金叉、KDJ底部、放量突破股

- **命中≥3个指标**：重点关注，可考虑介入
- **命中≥2个指标**：次级关注，等待确认
- **命中1个指标**：观察，不急于操作

## 四、龙头股评分模型

### 评分维度（总分100+）

| 维度 | 分值 | 说明 |
|------|------|------|
| 连板数 | 0-50 | 每连板+10分 |
| 封板时间 | 0-20 | 早盘封板加分 |
| 开板次数 | 0-20 | 0次满分，每开1次-5分 |
| 成交额 | 0-10 | 适中最佳 |
| 换手率 | 0-10 | 适中最佳 |
| 题材热度 | 加分项 | 热门题材+10分 |

### 龙头特征

- 连板高度最高
- 封板时间早且稳
- 题材空间大
- 市场辨识度高

## 五、板块强度分析

### 逆势上涨条件

```
板块涨幅 > 大盘涨幅
个股涨幅 > 板块涨幅
个股涨幅 > 0
```

**意义**：在大盘走弱时，逆势上涨的板块和个股往往是资金主攻方向

### 共振板块条件

```
大盘上涨时：板块涨幅 > 大盘涨幅 + 0.5%
大盘下跌时：板块跌幅 < 大盘跌幅 - 0.5%
```

**意义**：与大盘同向但更强的板块，说明资金集中流入

| 因子 | 权重 | 计算方式 |
|------|------|----------|
| 涨幅 | 20分 | 涨幅>7%满分 |

| 量比 | 10分 | 量比>2满分 |
| 弱转强 | 15分 | 低开高走满分 |

### 评分等级

- **80-100分**：极强，重点关注
- **60-79分**：较强，可考虑
- **40-59分**：一般，观望
- **<40分**：弱势，回避

## 七、技术指标信号

### MACD金叉
- 条件：涨幅1-8%，量比>1，收阳线
- 信号：趋势转多，可关注

- 条件：下影线占比>20%
- 信号：底部反转，超跌反弹

### RSI超卖
- 条件：涨幅>2%，量比>1.5，换手>2%
- 信号：超卖反弹，短线机会

### 突破形态
- 条件：涨幅>3%，量比>1.5
- 信号：放量突破，趋势启动

## 八、实战应用流程

### 每日复盘步骤

1. **判断情绪周期**
   - 查看涨停数、连板高度、涨跌比
   - 确定当前处于哪个周期阶段

2. **执行对应策略**
   - 根据周期选择关注方向
   - 回避该周期的风险操作

3. **筛选多指标共振股**
   - 查看命中≥3个指标的个股
   - 结合周期策略确定是否符合

4. **评估龙头股**
   - 关注龙头评分>60的个股
   - 分析连板高度和封板质量

5. **分析板块强度**
   - 找出逆势上涨板块
   - 关注共振板块龙头

6. **制定次日计划**
   - 确定关注池
   - 设定买入条件和止损位

## 九、风险提示

1. **情绪周期判断有滞后性**，需结合盘面实时调整
2. **多指标共振不代表必涨**，仍需关注大盘环境
3. **龙头股风险大**，高收益伴随高风险
4. **技术指标是辅助**，不能机械使用
5. **严格执行止损**，控制单次亏损在可承受范围内
